If the extension is not active

  1. Enable MapLink Router in Settings → Apps → Safari → Extensions.
  2. Set Website Access to All Websites (recommended).
  3. If using Selected Websites, allow https://maplinkroutertest.sigmology.ch explicitly.
  4. Test in a normal (non-Private) Safari tab first. Private Browsing may disable extensions unless enabled.
  5. If the Settings / Options page doesn't appear, force-quit Safari and reopen it, then tap Retry handshake.

Private Browsing note

Safari may disable extensions in Private Browsing. If you want MapLink Router there, enable Allow in Private Browsing in Settings.
